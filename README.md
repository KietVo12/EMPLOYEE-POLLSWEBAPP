## ğŸš€ Setup Instructions

### Requirements

- [Node.js](https://nodejs.org/) and npm or [pnpm](https://pnpm.io/)

### Steps to Install

1. Clone this repository:
   ```bash
   git clone https://github.com/KietVo12/EMPLOYEE-POLLSWEBAPP.git
   ```
2. Navigate to the project directory:
   ```bash
   cd EmployeePollsWebApp
   ```
3. Install all necessary dependencies:
   ```bash
   pnpm install
   ```

### Run the Development Server

1. Start the server locally:
   ```bash
   pnpm dev
   ```
2. Access the application at [http://localhost:5173](http://localhost:5173).
